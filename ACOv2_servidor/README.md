# Algoritmo de Colonias de Hormigas: Servidor del robot.

_Código en C++ referente al servidor realizado en Qt._

## Descripción de archivos del directorio

_Descripción de los archivos del proyecto servidor disponibles en el documento 'Gomez-Hutchison-JuanManuel_TFG' en la Sección 3.8.3._


### Pre-requisitos 

_Que cosas necesitas para instalar el software:_

* QtCreator v4.4.10 o superior.

### Instalación 

_Cargar el programa en el IDE del QtCreator y generar un archivo ejecutable o bien ejecutarlo desde el mismo IDE._

## Ejecutando las pruebas 

_Para la ejecución debemos arrancar el servidor, posicionar sobre el nodo azul del mapa al robot, encenderlo, y pulsar sobre el botón 'Init pheromones' en el programa servidor­._
