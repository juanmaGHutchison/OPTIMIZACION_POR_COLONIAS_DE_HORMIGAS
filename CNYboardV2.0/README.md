# Diseño de Circuito Impreso para sensores CNY70.

_Proyecto realizado en Eagle donde se muestran los esquemáticos y el diseño de dicho esquemático donde se interconectan los sensores ópticos infrarrojos CNY70._

## Descripción de archivos del directorio

_Descripción de los archivos principales de este directorio:_

* 'CNYboard.brd': Diseño del circuito impreso.
* 'CNYboard.pdf': PDF donde se muestra el diseño realizado en el archivo 'CNYboard.brd'.
* 'CNYboard.sch': Esquemático del circuito impreso.
* 'CNYboard.pdf': PDF donde se muestra el esquemático realizado en el archivo 'CNYboard.sch'.
* 'eagle.epf': Archivo de tipo proyecto, el cual abrira los archivos en el programa Eagle.


### Pre-requisitos 

_Que cosas necesitas para instalar el software:_

* Autodesk Eagle en su última versión.


## Ejecutando las pruebas 

_PCB totalmente testeada tanto por software con la herramienta DRC, tanto impresa, de manera empírica._
