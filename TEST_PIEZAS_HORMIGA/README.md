# Algoritmo de Colonias de Hormigas: Pruebas para el Robot en Arduino.

_Código en Arduino(C++) para probar los componentes electrónicos del robot tanto individualmente como algunas pruebas de integración._

## Descripción de archivos del directorio

_Descripción del archivo principal de este directorio. El resto de archivos podemos encontrarlos descritos en el documento 'Gomez-Hutchison-JuanManuel_TFG' en la Sección 3.3.7:_

* 'TEST_PIEZAS_HORMIGA.ino': Código principal en Arduino, el cual cargaremos en el robot físicamente.


### Pre-requisitos 

_Que cosas necesitas para instalar el software:_

* Arduino IDE en su última versión.

### Instalación 

_Cargamos el código en el puerto serie al que esté conectado el robot al PC._

## Ejecutando las pruebas 

_Para probar las piezas, una vez montado el robot iremos descomentando las directivas de preprocesador escritas al principio del programa, y subiremos el código. Cada una de estas directivas vienen descritas._
