# Diseño del mapa sobre el que se ejecutará el robot.

_Mapa que representa el grafo sobre el que el robot en Arduino realizará la ejecución. Se imprimirá dicho diseño en papel Din A3._

## Descripción de archivos del directorio

_Descripción de los archivos principales de este directorio:_

* 'mapaV4.svg': archivo para editar el mapa.
* 'mapaV4.png': foto del archivo del mapa.
* 'mapaV4.pdf': archivo pdf que contiene la foto del mapa, perfecto para imprimir.
