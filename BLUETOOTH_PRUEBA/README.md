# Algoritmo de Colonias de Hormigas: Pruebas para la comunicación del Robot en Arduino.

_Código en Arduino(C++) para probar la comunicación Bluetooth del robot._

## Descripción de archivos del directorio

_Descripción del archivo principal de este directorio. El resto de archivos podemos encontrarlos descritos en el documento 'Gomez-Hutchison-JuanManuel_TFG' en la Sección 3.3.7:_

* 'BLUETOOTH_PRUEBA.ino': Código principal en Arduino, el cual cargaremos en el robot físicamente.


### Pre-requisitos 

_Que cosas necesitas para instalar el software:_

* Arduino IDE en su última versión.

### Instalación 

_Cargamos el código en el puerto serie al que esté conectado el robot al PC._

## Ejecutando las pruebas 

_Con el Módulo Bluetooth HC-05 conectado al robot, y el servidor en ejecución, cargamos el programa y realizamos una transmisión con el botón 'Start Transmission'­._
